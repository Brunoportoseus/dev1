# Como adicionar as fotos dos produtos

O catálogo já está pronto. Basta colocar as fotos **nesta pasta** (`images/`)
com **exatamente os nomes abaixo**. Se uma foto não for adicionada, o card
mostra automaticamente a ilustração elegante (o catálogo nunca quebra).

## Nomes exatos dos arquivos

| Produto              | Código              | Nome do arquivo          |
|----------------------|---------------------|--------------------------|
| Kitten 64            | ZP5805              | `kitten.jpg`             |
| Paio 64              | ZP5426              | `paio.jpg`               |
| Shell 64             | ZP4318              | `shell-64.jpg`           |
| Shell 48             | ZP4319              | `shell-48.jpg`           |
| Pipe Simétrico       | ZP5780              | `pipe-simetrico.jpg`     |
| Pipe Direcional      | ZP5780E / ZP5780D   | `pipe-direcional.jpg`    |
| Plymouth             | ZP5420              | `plymouth.jpg`           |
| Garage               | ZP1254              | `garage.jpg`             |

## Regras importantes

1. **O nome precisa ser idêntico** (tudo minúsculo, sem acento, sem espaço).
2. Use a extensão **`.jpg`**. Se a sua foto for `.png`, renomeie para `.jpg`
   (ou me avise que eu troco no código).
3. Formato recomendado: **horizontal**, proporção aproximada **3:2 ou 4:3**
   (ex.: 1200×800 px). Fundo claro fica melhor.
4. Tamanho ideal: até ~500 KB por imagem (para abrir rápido no WhatsApp).

## Passo a passo pelo site do GitHub (funciona no celular e no PC)

1. No repositório, abra a pasta **`images`**.
2. Clique em **`Add file`** → **`Upload files`**.
3. Arraste todas as fotos (já renomeadas conforme a tabela acima).
4. Em "Commit changes", clique no botão verde **`Commit changes`**.
5. Aguarde ~2 minutos. O catálogo atualiza sozinho com as fotos reais.

Pronto! As fotos aparecem automaticamente no lugar das ilustrações.
