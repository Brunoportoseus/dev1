# Frameworks Estratégicos

## Estratégia e Posicionamento

| Framework | Quando usar |
|-----------|-------------|
| SWOT | Análise inicial de projeto/mercado |
| Porter's 5 Forces | Análise competitiva profunda |
| Blue Ocean Strategy | Quando precisa se diferenciar radicalmente |
| Value Proposition Canvas | Definir proposta de valor |
| Jobs To Be Done | Entender o que o cliente quer resolver |
| Ansoff Matrix | Decisão de crescimento (novo produto/mercado) |
| BCG Matrix | Priorizar produtos no portfólio |

## Funil e Conversão

| Framework | Quando usar |
|-----------|-------------|
| AIDA | Estrutura macro de qualquer peça |
| PAS (Problem-Agitate-Solve) | Copy focada em dor |
| Hook-Story-Offer | Funis de lançamento (Russell Brunson) |
| Value Ladder | Montagem de esteira de produtos |
| Awareness Levels (Schwartz) | Definir tom/ângulo por estágio do público |
| Pirate Metrics (AARRR) | Growth: Aquisição-Ativação-Retenção-Receita-Referência |
| ROAS/LTV Analysis | Viabilidade financeira da operação |
| Offer Creation (Hormozi) | Montagem de oferta irrecusável |

## Pesquisa

| Framework | Quando usar |
|-----------|-------------|
| Voice of Customer (VoC) | Coletar linguagem real do público |
| Mining de reviews | Extrair dores/desejos de avaliações |
| Competitor Funnel Map | Mapear funis dos concorrentes |
| Customer Journey Map | Entender toda a jornada do cliente |
| RFM Analysis | Segmentar base existente |

## Princípios dos Mestres de Copy

| Mestre | Princípio |
|--------|-----------|
| David Ogilvy | Respeite o leitor. Fatos > adjetivos. Headlines irresistíveis |
| Eugene Schwartz | Canalize desejos que já existem. Níveis de consciência |
| Gary Halbert | Lista faminenta primeiro. Escreva como para um amigo. Especificidade |
| Claude Hopkins | Publicidade científica. Dê razões específicas |
| John Caples | Headline com benefício específico. Teste tudo |
| Robert Collier | Entre na conversa que já está na mente do prospect |
| Joseph Sugarman | Escorregador de sabão: cada frase faz ler a próxima |
| Dan Kennedy | Escassez autêntica. Qualifique o público. Marketing mensurável |
| Gary Bencivenga | Bullets hipnóticos. Provas irrefutáveis. Unique mechanism |
| Victor Schwab | 5 passos: atenção → vantagens → provas → emoção → ação |
| Alex Hormozi | Oferta tão boa que a pessoa se sente burra dizendo não |
| Russell Brunson | Hook-story-offer. Construa a crença antes de pedir a venda |
| Ben Settle | E-mails diários que entretêm E vendem. Personalidade acima de tudo |
