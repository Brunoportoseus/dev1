Como as alterações funcionam?

As alterações feitas aqui ficam salvas no navegador (localStorage). Para que todos os clientes vejam as mudanças:

  1. Clique em Exportar JSON para baixar o arquivo catalog-data.json
  2. Acesse o repositório no GitHub
  3. Suba o arquivo catalog-data.json na pasta raiz do projeto
  4. Aguarde ~2 minutos para o site atualizar

Como adicionar fotos dos produtos?

No campo "Arquivo de Foto" de cada produto, coloque o nome do arquivo (ex: kitten.jpg). Depois:

  1. Renomeie a foto com exatamente esse nome
  2. Acesse o GitHub → pasta images/
  3. Clique em Add file → Upload files
  4. Faça o upload e clique em Commit changes

Se a foto não for encontrada, a ilustração original aparece automaticamente.
