Para que serve?

Com o token configurado, você pode enviar fotos e publicar alterações diretamente daqui, sem precisar acessar o GitHub manualmente.

Como criar o token:

  1. Acesse github.com/settings/tokens
  2. Clique em Generate new token (Fine-grained)
  3. Em Repository access, selecione apenas brunoportoseus/dev1
  4. Em Permissions → Contents, escolha Read and write
  5. Clique em Generate token e copie o valor

O token é salvo apenas neste navegador (localStorage). Nunca é enviado para terceiros.

Segurança

Apaga todas as alterações e volta ao catálogo original.

Fluxo de trabalho

  1. Edite os produtos na aba Produtos
  2. Envie as fotos na aba Fotos
  3. Clique em 🚀 Publicar no GitHub — isso atualiza o site para todos
  4. Aguarde ~2 minutos e o catálogo aparece atualizado

O botão Publicar requer o token do GitHub (configure em Configurações).
